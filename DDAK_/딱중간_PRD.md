# 딱중간 PRD v1.0

> 지하철 소요시간 기반 중간지점 찾기 서비스

---

## 1. 서비스 개요

여러 출발지에서 지하철로 이동할 때, 모두에게 가장 공평한 중간 지점을 찾아주는 서비스.
기존 서비스(씩 등)와 달리 **지리적 중간점이 아닌 실제 지하철 소요시간 기준**으로 계산한다.

---

## 2. 핵심 로직

### 2-1. 중간역 후보 선정
1. 입력된 출발지 역들의 좌표 **무게중심(평균)** 계산
2. 무게중심 기준 반경 **5km** 내 수도권 지하철역 전부 후보로 추출
3. 5km 내 후보 없으면 **10km**로 자동 재시도
4. 10km도 없으면 결과 없음 처리

### 2-2. 공평도 점수 (0~100)
- 각 출발지 → 후보역 지하철 소요시간 계산
- 환승 1회당 소요시간 **1.5배** 페널티
- 동일역 출발 인원수만큼 **가중치 반영** (최대 2배)
- 점수 산출:
  ```
  편차 페널티 = (최대 소요시간 - 최소 소요시간) * 가중치A
  총합 페널티 = 전체 소요시간 합 / 정규화값 * 가중치B
  공평도 = 100 - 편차 페널티 - 총합 페널티
  ```
- 가중치A, 가중치B는 상수로 분리해 추후 튜닝 가능하게 구현

### 2-3. 번화함 점수 (0~100)
- 카카오 로컬 API, 후보역 기준 반경 **500m**
- `meta.total_count` 기준으로 카운트
  ```
  번화함 = 식당수(FD6) * 0.4 + 카페수(CE7) * 0.4 + 주점수(PO3) * 0.2
  ```
- 전체 후보역 중 정규화해서 0~100으로 환산

### 2-4. 최종 점수
```
최종 점수 = 공평도 * 0.7 + 번화함 * 0.3
```
- 최종 점수 기준 내림차순 정렬
- 가중치 슬라이더는 v2로 이월

## 3. 입력 스펙

| 항목 | 내용 |
|------|------|
| 출발지 수 | 최소 2개, 최대 10개 |
| 입력 방식 | 지하철역명 텍스트 입력 + 자동완성 |
| 동일역 처리 | 인원수 설정 UI 노출 (기본 1명, 가중치 최대 2배) |
| 조회 시간 | 토글 ON: 현재시간 기준 / 토글 OFF: 날짜+시간 직접 설정 |
| 범위 | 수도권 전철 전체 |

---

## 4. 결과 스펙

| 항목 | 내용 |
|------|------|
| 기본 노출 | TOP 3 |
| 더보기 | 최대 10개까지 |
| 최소 노출 | 1개 이상이면 있는 만큼 노출 |
| 결과 없음 | "앗 딱중간이 없어요. 가위바위보해서 지는 사람이 가기~" |

### 결과 카드 표시 항목
- 역명 + 호선 뱃지
- 공평도 점수
- 각 출발지별 소요시간 (이모지 + 역명 + 분)
- 환승 횟수 표시
- 번화함 지표: 🍽 식당 N곳 · ☕ 카페 N곳 · 🍺 주점 N곳

---

## 5. 예외처리 정책

| 케이스 | 처리 방식 |
|--------|----------|
| 특정 출발지 API 에러 | 해당 출발지만 제외하고 나머지로 계산 + 인라인 배너 안내 |
| 전체 API 오류 | "잠깐, 길을 잃었어요. 잠시 후 다시 시도해줘" + 재시도 버튼 |
| 5km 내 후보역 없음 | 10km로 자동 재시도 + "10km로 넓혀서 찾았어요" 노티스 |
| 10km도 없음 | 결과 없음 문구 처리 |
| 결과 1~2개 | 있는 만큼 노출 |

---

## 6. 화면 목록

1. **홈 화면** — 출발지 입력 + 조회시간 토글 + CTA
2. **로딩 화면** — 스피너 + 랜덤 로딩 문구
3. **결과 화면** — 노선도 스타일 지도 + 결과 카드 TOP3
4. **더보기 상태** — 4~10위 카드 슬라이드 다운
5. **결과 없음 화면** — 에러 문구
6. **일부 에러 상태** — 인라인 배너 + 정상 결과 카드
7. **전체 에러 화면** — 재시도 버튼
8. **반경 확장 알림** — 결과 상단 인라인 노티스
9. **공유 바텀시트** — 카카오톡 / 링크 복사 / 이미지 저장

---

## 7. 사용 API

### 공공데이터포털
- **서울특별시 대중교통환승경로 조회 서비스**
- 엔드포인트: `http://ws.bus.go.kr/api/rest/pathinfo/getPathInfoBySubway`
- 용도: 출발지 → 후보역 지하철 소요시간 + 환승 횟수

### 카카오 로컬 API
- **키워드 검색**: 역명 자동완성 + 좌표 조회
- **카테고리 검색**: 반경 500m 식당/카페/주점 카운트
- Base URL: `https://dapi.kakao.com/v2/local`
